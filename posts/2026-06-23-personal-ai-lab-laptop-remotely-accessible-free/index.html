I have an AI personal assistant, an AI coding agent, and a collection of personal projects — all running locally on my laptop. And I can reach every single one of them from my phone, from anywhere in the world, without paying for a single cloud server.

No VPS. No AWS bill. No Docker Compose on some remote VM. Just my laptop at home, a free VPN, and a handful of open-source tools running on ports. Here's exactly how it works — and how you can replicate it.

Everything in this stack is free. The only optional cost is the AI model — and even heavy usage runs for a fraction of a cent per day with DeepSeek.

The Stack

  • OpenClaw — a local AI personal assistant with a web UI. Chat with it, ask it to check emails, delegate tasks. Runs entirely on your laptop.
  • OpenCode — an AI coding agent. Runs in your terminal, exposes a browser UI for reading and editing code with AI assistance.
  • Rover — an open-source remote command runner and project launcher I built. Run any shell command from your phone, start and stop registered projects, stream live logs.
  • Tailscale — free private networking over WireGuard. Creates an encrypted tunnel between your devices. No ports open to the internet.
  • DeepSeek — the AI model powering the tools. Free tier is generous; paid usage is extremely cheap.

The Core Idea

Every tool in this stack is a web server listening on a port. Your laptop is the server. Your phone is the client. Tailscale is the private network connecting them — encrypted end-to-end, invisible to the internet.

From your phone, you open http://100.x.x.x:PORT — your laptop's Tailscale IP and the port the tool runs on. The request travels through a WireGuard tunnel directly to your laptop. Nobody on the internet can see your services. Nobody can reach those ports except devices logged into your Tailscale account.

Your laptop already has more CPU, RAM, and storage than any cloud VM you'd rent. The only missing piece was access — and Tailscale solves that for free.

Step 1 — Install Tailscale (5 Minutes)

Tailscale uses WireGuard under the hood: fast, modern, battle-tested cryptography. Setup is genuinely five minutes.

  1. Create a free account at tailscale.com — sign in with Google or GitHub
  2. Install Tailscale on your laptop — tailscale.com/download
  3. Install the Tailscale app on your phone from the App Store or Play Store
  4. Log into the same Tailscale account on both devices

Both devices appear in your Tailscale admin console. Find your laptop's Tailscale IP — it starts with 100. Every service you start on your laptop is now reachable from your phone at that IP.

Step 2 — OpenClaw: Your AI Personal Assistant

OpenClaw is a local AI assistant with a web interface. Chat with it the same way you'd chat with any AI assistant — but it runs on your machine, and your conversations stay private.

openclaw gateway start

OpenClaw starts on port 3000. From your phone:

http://100.x.x.x:3000

In settings, set the model to DeepSeek. It handles everyday tasks well and costs almost nothing. If you need more reasoning power, DeepSeek R1 is available in the same tier.

Step 3 — OpenCode: Your AI Coding Agent

OpenCode is an AI coding agent that runs in your terminal and exposes a browser UI. From that UI you can browse your codebase, make edits, and get AI-driven suggestions — without touching your keyboard.

# Install
npm install -g opencode

# Start the web UI on port 4000
opencode --port 4000

Access it from your phone at http://100.x.x.x:4000. You can make real code changes on a real codebase from your phone. Pair it with DeepSeek as the model for near-zero cost.

I've shipped production fixes from my phone during a commute. OpenCode on a mobile browser is genuinely usable for real work.

Step 4 — Rover: Remote Commands and Project Launcher

Rover is the glue. I built it to solve one specific problem: what do you do when something crashes and you're not home?

With Rover you can:

  • Run any shell command from your phone browser — openclaw gateway restart, git pull, anything
  • Register your projects and start or stop them with a single tap
  • Stream live console output to your phone in real time
  • Keep a scrollable history of past sessions with full output

Rover is open source at github.com/ylnhari/rover. Clone it, build it, use it:

git clone https://github.com/ylnhari/rover
cd rover
go build -o rover .

# Generate a strong secret and start
export ROVER_SECRET=$(openssl rand -hex 32)
echo $ROVER_SECRET   # save this — you need it to log in
./rover serve --addr :2278

Open http://100.x.x.x:2278 on your phone, log in, and you're in. Dark terminal-style UI, real-time command output, Projects tab for your registered services.

Adding a project to Rover

Projects tab → Add Project → pick a directory and start file (.py, .sh, .js, .go, etc.) → Validate & Add. Rover test-runs it, detects the URL it opens, and saves it. One tap to start. One tap to stop. Live logs streaming to your phone.

Keep Everything Running When You Leave

Start your services in the background before walking out:

# Rover first — it's the recovery tool for everything else
ROVER_SECRET=<your-secret> ./rover serve --addr :2278 &

# AI assistant
openclaw gateway start &

# Coding agent
opencode --port 4000 &

If anything crashes while you're away: open Rover → Terminal → restart command. Full recovery without touching your laptop.

Security

Tailscale means none of these ports are reachable from the internet. Only devices authenticated to your Tailscale account can see them. Three things to do on top of that baseline:

  • Enable 2FA on your Tailscale account. Non-negotiable. If your account is compromised, an attacker is inside your network. 2FA shuts that down in two minutes.
  • Set --secret on Rover. Rover runs shell commands on your machine. Even inside a private network, a secret means a briefly unattended phone is useless to an attacker.
  • Know how to remove a device. Phone lost or stolen? Tailscale admin console → remove it. Access revoked immediately.
